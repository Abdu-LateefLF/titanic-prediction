# Random Forest Titanic Analysis

This workspace contains a Jupyter notebook for exploring and modeling the Titanic dataset using a random forest approach.

## Files

- `random-forest.ipynb` — analysis notebook for data exploration, preprocessing, model training, and evaluation.
- `Titanic-Dataset.csv` — dataset used in the notebook.

## Dataset source

The dataset was obtained from Kaggle:
https://www.kaggle.com/datasets/yasserh/titanic-dataset
